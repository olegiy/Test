# Paleograph Manuscript Analyzer

An intelligent web application for analyzing, transcribing, and structuring text from historical manuscripts using Google's Gemini AI.

## Features
- **Double-Pass OCR:** High-precision transcription using ensemble variants.
- **Structural Integrity:** Preserves tables and registers in digital formats.
- **Local-First:** Privacy-focused processing in your browser.
- **Premium Design:** Scholarly aesthetic with intuitive controls.

## Getting Started
1. Clone the repository.
2. Run `npm install`.
3. Set up your Gemini API key in `.env`.
4. Run `npm run dev`.
